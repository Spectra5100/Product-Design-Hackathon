# Xano endpoints (Backend Adrien et alicia)

## Operators
- GET /operator
Retourne la liste des opérateurs (RATP, TCL, RTM)

## Metrics
- GET /operator_metrics
Retourne les indicateurs environnementaux, liés via operator_id

## Champs clés
operator_metrics : co2, energy, clean_share_pct, avg_occupancy_rate, global_score, year, source
