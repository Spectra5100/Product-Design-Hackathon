# Données Sources

Ce dossier contient les fichiers CSV originaux qui ont été importés dans Xano.

## Fichiers

- `dbo-operator-*.csv` : Liste des opérateurs de transport
- `dbo-operator_metrics-*.csv` : Métriques environnementales

## Import dans Xano

1. Se connecter à Xano
2. Database > Sélectionner la table
3. Actions (⋮) > Import Data
4. Sélectionner le fichier CSV
5. Mapper les colonnes
6. Importer

## Format des CSV

Les CSV doivent avoir des en-têtes correspondant aux colonnes de la base de données.
